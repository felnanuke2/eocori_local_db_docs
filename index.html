local_db

Local Database

A Cache database for Ecomission App

Motivation

For cache all data obtained from SDK this package is needed.

Installation

You can import this package add

local_db:
      path: ../local_db

on your pubspec.yaml.

Getting Started

Import on you dart file

import 'package:local_db/local_db.dart';

See the then you can instantiate te local_db with isar isar

final database = IsarDatabase();

await database.init(dbPath: path);

or create your own implementing IDatabase like

class IsarDatabase implements IDatabase {}
